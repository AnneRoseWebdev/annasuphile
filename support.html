FAQ

Q : Comment installer BombiTMan ?

R : Veuillez vous référer au guide d’installation sur la page Download pour obtenir des instructions.

Q : J’ai des problèmes techniques avec le jeu. Que dois-je faire ?

R : Consultez notre section de dépannage ci-dessous pour connaître les problèmes courants et les solutions. Si votre problème persiste, veuillez contacter notre équipe d’assistance.

Q : Puis-je jouer à BombiTMan sur des appareils mobiles ?

R : Actuellement, BombiTMan n’est disponible que pour les plates-formes de bureau (Windows, MacOS, Linux).

Dépannage

Le jeu se bloque au démarrage

Si le jeu plante immédiatement au démarrage, essayez les étapes suivantes :

  1. Assurez-vous que vous avez bien fait un "pip install pygame" dans votre terminal sinon le jeu ne se lancera pas
  2. Vérifiez l’intégrité des fichiers du jeu via le lanceur (par tous les images et que aucun fichier n'est manquant).
